**NXT RoboEyes**
Портированная библиотека RoboEyes для конструктора LEGO Mindstorms NXT (использующая nxt-python). Позволяет создавать выразительные анимированные глаза для вашего робота на стандартном дисплее NXT.

Этот код является адаптацией версии для MicroPython от MCHobby, которая, в свою очередь, основана на оригинальной Arduino-библиотеке от FluxGarage.

Особенности порта для NXT
Оптимизация под экран NXT: Поддержка разрешения 100x64 пикселя.

Прямой доступ к памяти (IOMap): Для обеспечения плавной анимации используется запись в системную область памяти дисплея (0xA0001), что значительно быстрее стандартных методов отрисовки.

Геометрические примитивы: Включает кастомные реализации отрисовки закругленных прямоугольников и треугольников, оптимизированные под структуру буфера экрана NXT.

Система настроений: Поддержка состояний: DEFAULT, TIRED, ANGRY, HAPPY, FROZEN, SCARY, CURIOUS.

Установка
Убедитесь, что у вас установлена библиотека nxt-python.

Скопируйте файл nxt_roboeyes.py в директорию вашего проекта.

Быстрый старт
Python

import nxt.locator
from nxt_roboeyes import NxtDisplay, RoboEyes, HAPPY

# Подключение к кирпичу NXT
brick = nxt.locator.find_one_brick()

# Инициализация драйвера дисплея и библиотеки глаз
disp = NxtDisplay(brick)
eyes = RoboEyes(disp, width=100, height=64)

# Настройка
eyes.set_auto_blinker(True) # Включить автоматическое моргание
eyes.mood = HAPPY           # Установить счастливое настроение

# Основной цикл анимации
while True:
    eyes.update() # Рассчитывает кадр и отправляет его на дисплей
Основные возможности управления
Настроения (Moods)
Вы можете динамически менять выражение глаз робота:

eyes.mood = TIRED — веки опускаются, создавая сонный вид.

eyes.mood = ANGRY — веки наклоняются к центру.

eyes.mood = HAPPY — нижние веки приподнимаются.

Анимации
eyes.blink() — принудительное моргание.

eyes.wink(right=True) — подмигивание правым глазом.

eyes.confuse() — анимация "замешательства" (тряска глаз по горизонтали).

eyes.laugh() — анимация смеха (тряска по вертикали).

Поведение
eyes.set_idle_mode(True) — робот начнет случайно оглядываться по сторонам.

eyes.cyclops = True — режим циклопа (один большой глаз по центру).

Технические детали отрисовки
Поскольку дисплей NXT монохромный, библиотека использует BGCOLOR = 0 (белый/пустой) и FGCOLOR = 1 (черный). Обновление экрана происходит пачками по 40 байт через IOMap для минимизации задержек интерфейса USB/Bluetooth.
